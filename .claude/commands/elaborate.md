# Elaboration 
推敲に使うプロンプト

## プロンプト
```
直訳で脳神経科学の研究者(とくに言語学と脳内の言語処理基盤の専門家)がわかるような学術的な英語に直してください。 ただし以下のルールに従ってください:
遠回しな表現は使わないでください。
主語が明らかではないところは明確化させて、できるだけ能動態を用いてください。
ただし主体が推論できないところは受動態でかまいません。
接続詞の利用は文章の流れが変わるときだけ利用するようにして必要最低限になるようにしてください。
Thisなど、不明瞭となりえる代名詞の利用は避けてください
研究はstudy, 先行研究はprior studyとしてください。
同じ主語が複数のセンテンスで連続する場合は体裁を整えてください。
時制はConclusion以外では過去形は使わないでください。
研究はstudyで表してください。
提案手法はproposed protocolとしてください。ー＞ 修正
「」やクオーテーションは``''としてください。
章はすべてsectionです。
the paperと訳せる部分はすべてthe studyに変えてください。
先行研究の手法、に言及する際はapproachという単語を使ってください not method.
フォロワーやリーダーにはノードという単語がついていない場合はつけてください。
表現のずれがあったら適切な語に一致させてください。
固有名詞については、固有名詞で表現しなくても良い部分に関しては普通の名詞として使ってください。
単語は前に訳した単語とできるだけ一致させてください（意図が変わらなければ）。
記号の表記が相応しいところがあれば付与してください。（例：新しいビュー\mathcal{V}_{new}となっているが、あるところでは新しいビューは、のように記号が抜けている）
図やセクションなどにラベルが振られていない場合は、振ってください。
結果は容易にコピーできるようにLaTeXのソースコードを出力してください。
```
