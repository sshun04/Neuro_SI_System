# Spell Check
スペルチェックに使うプロンプト

## プロンプト
```
誤字がないかチェックしてください。
・スペルミスがないか
・脱字がないか
・センテンスは文法的にcorrectか
・不要なところが大文字になっていないか
・文末にドットはついているか
・動詞や名詞など、表現にずれがないか
・時制は統一されているか（結論以外は原則現在形）
・単語の前後には半角スペースを入れていか
・すべての文字が半角文字であり、全角文字（全角スペースを含む）は使われていないか
・単語の省略表現に使うピリオドの後ろは半角スペース 1 つであるか
・数字はmathmodeで表現されているか
・mathmodeのとき、改行は\columnsizeによって自動的にされるようになっているか
・短く言える場合はできる限り短い単語を使っている。同じ意味の簡単な単語があるならば、そちらをつかう。類語重複をなくす。絶対的な意味を持つ単語を修飾しない。たとえばcheck(into)、later (on), climb (up), link (up), close (down), open (up), dvide (up), seal(off), end (up)などの()は不要。it is clear that → clearlyなどのシンプル化。
・and, but, so を文頭で使ってはいけない。口語的表現であるため
・et al.」の前のカンマの使い方が正しいか
・代名詞や there を使いすぎていないか
・形容詞（long、 heavy、fast など）や形容動詞（beautiful、fresh など）を使うときには量をはっきりさせているか？
・略語に冠詞をつけるとき、略語の発音に応じて a と an を使い分けているか？
・冠詞の付け方に乱れはないか？
・修飾語、形容詞、副詞の修飾先がはっきりしているか？
・it が文章中の何を受けるのかがはっきりわかるように使われているか？
・主語と述語がちゃんと対応しているか？（倫理的、表現的におかしくないか）
・英文が \Simple is best" の原則に基づいているか？
・対照や比較を示す言葉としては while ではなく whereas を使っているか？
・since や as を because の意味で使っていないか。その意味で使うならばすべて because を使う。
・三単現の「s」が落ちていないか？
```
